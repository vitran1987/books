## QUALITY STANDARDS

### All data must have:
- ✅ Primary source identified
- ✅ Date of data specified
- ✅ Multiple sources if possible (cross-verification)
- ✅ Screenshots/archives for web sources
- ✅ No Wikipedia as sole source (use Wikipedia's sources instead)

### Red flags to avoid:
- ❌ "According to various sources" (be specific)
- ❌ Outdated data (>2 years old for market data)
- ❌ Unverified numbers (can't find original source)
- ❌ Contradicting data (resolve before using)
