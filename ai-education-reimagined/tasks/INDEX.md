## How to Use These Files

### For Each Chapter Task File

Each chapter task file contains:

1. **Chapter Overview**: Focus, subsection count, timeline, word count
2. **Phase 1: Research (Weeks 1-3)**: Detailed research tasks, sources, data collection
3. **Phase 2: Writing (Weeks 4-10)**: Week-by-week subsection writing with specific requirements
4. **Phase 3: Review (Weeks 11-12)**: Verification, quality checks, polish
5. **Deliverables Checklist**: Files to create, supporting materials, quality metrics
6. **Resource Links**: Where to find key sources
7. **Success Criteria**: What makes this chapter work or fail

### Workflow Recommendation

1. **Start with research phase**: Complete all research before writing
2. **Write subsections sequentially**: One per week, following task file guidelines
3. **Review as you go**: Don't wait until end to verify sources
4. **Track completion**: Check off tasks as completed
5. **Maintain quality**: Every subsection must meet success criteria

## Notes and Reminders

### Critical Requirements (Every Chapter)

1. **Pure Vietnamese**: NO English/Vietnamese mixing
2. **Story-driven**: Long narrative paragraphs, NOT bullet points
3. **Engaging openings**: NOT time-based patterns
4. **Evidence-based**: Every claim backed by credible source
5. **Vietnamese context**: Integrate local examples throughout
6. **Accessible**: 10th grader should understand
7. **Practical**: Actionable insights in every section

### Reference Writing Style
**MUST study**: `blockchain-ico\chapter-01\01-lich-su-va-thanh-cong-ico.md`

This reference demonstrates:
- Long narrative paragraphs (150-300 words)
- Story-driven openings (character/situation/drama)
- Detailed case studies with specific metrics
- Business context before technical details
- Statistics integrated naturally
- Conversational yet authoritative tone
- Multiple perspectives (success AND failure)
- Actionable insights at section ends
